# Crypto Price Alerts

## Description
Monitors cryptocurrency prices (BTC, ETH, etc.) using the CoinGecko or Binance API. It checks the price against a user-defined threshold and sends an immediate alert to Telegram or Discord when a price target is hit.

## Features
- **Price Monitoring:** Fetches real-time data every 10 minutes.
- **Conditional Alerting:** Only notifies if the price change percentage exceeds a specific limit.
- **Multi-Channel:** Supports Telegram bot integration.
